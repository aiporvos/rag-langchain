# Manual de Procedimientos de Intercambio\n\n## 1. Validación de Fondos\nAntes de procesar cualquier orden, el sistema verifica la disponibilidad de fondos en la red del Bridger. Este proceso toma microsegundos gracias a nuestra API WebSocket.\n\n## 2. Ejecución de Swap\nEl intercambio se realiza mediante un Smart Matcher que busca la tasa más competitiva entre 15 proveedores de liquidez distintos.\n\n## 3. Confirmación de Red\nPara transacciones de Bitcoin, esperamos 3 confirmaciones. Para Ethereum y redes Layer 2 como Polygon o Arbitrum, esperamos 12 confirmaciones de seguridad.\n\n## 4. Notificaciones\nEl usuario recibe alertas vía Push, Email y SMS una vez que la transacción alcanza el estado "Completada".
